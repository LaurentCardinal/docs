<Note>**Prerequisite** You need more than one rendering or photo of your project to enable this feature. please read the [checklist](/checklist).</Note>


<Steps>
  <Step title="Step 1">
    Access the configuration module within your project dashboard and navigate to the **Exterior** tab.
    ![360 1](/Users/juancamilocardonab/Desktop/V2 KB PICS/360 1.jpg) 
  </Step>
  <Step title="Step 2">
    Arrange the desired views in the preferred sequence for presentation within your project. Simply click on the **+** button to incorporate various views into the display order.
    ![360 2](/Users/juancamilocardonab/Desktop/V2 KB PICS/360 2.jpg)
  </Step>
  <Step title="Step 3">
    After organizing the images, you need to draw the simple polygons on each floor of each render. Just click on **Draw** or **Redraw** and repeat this process until finished.
    ![360 2](/Users/juancamilocardonab/Desktop/V2 KB PICS/360 3.jpg)
  </Step>
</Steps>

You can navigate the different views just by clicking here.

## Troubleshooting

Here's how to solve some common problems when setting up a 360º project.

<AccordionGroup>
  <Accordion title="Planpoint is not loading">
    Update to Node v18. Run `mintlify install` and try again.
  </Accordion>
  <Accordion title="No such file or directory on Planpoint">
Go to the `C:/Users/Username/.mintlify/` directory and remove the `mint`
folder. Then Open the Git Bash in this location and run `git clone
https://github.com/mintlify/mint.git`.

Repeat step 3.

  </Accordion>
  <Accordion title="Getting an unknown error">
    Try navigating to the root of your device and delete the ~/.mintlify folder.
    Then run `mintlify dev` again.
  </Accordion>
</AccordionGroup>
