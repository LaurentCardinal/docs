## Endpoints

The Planpoint API features endpoints for every common items of the Planpoint Viewer.

```JavaScript
api.schedule(function(done) {
  performCalculations();
  done();
}, "when-idle");
```

## Fetching a project's data

This is a small description.

```JavaScript
  async function fetchProject() {
      try {
        const response = await axios({
          method: 'POST',
          headers: {
            'content-type': 'application/json'
          },
          data: {
            namespace: namespace,
            hostName: hostName
          },
          url: '/api/projects/find'
        })

        setProject(response.data)
      } catch (err) {
        console.error(err)
      }
    }
```
